#### Описание

Данный проект представляет собой простое консольное приложение на языке Python для управления списком задач. Программа позволяет добавлять новые задачи, просматривать текущий список дел и отмечать выбранные задачи как выполненные. Список задач сохраняется в файл tasks.json в формате JSON и автоматически загружается при следующем запуске программы, что делает приложение удобным для регулярного использования.

Основной файл проекта — project.ipynb. В этом блокноте находятся:

- функция main, которая отвечает за запуск программы и отображение текстового меню;
- функция load_tasks, которая загружает список задач из файла tasks.json;
- функция save_tasks, которая сохраняет текущий список задач в файл;
- функция add_task, которая запрашивает у пользователя данные новой задачи и добавляет её в список;
- функция list_tasks, которая выводит на экран все задачи с их статусами и приоритетами;
- функция complete_task, которая позволяет пользователю отметить выбранную задачу как выполненную.

Я выбрала текстовый интерфейс, потому что он не требует установки дополнительных библиотек и хорошо демонстрирует базовые конструкции Python: функции, условия, циклы и работу со структурами данных (списки и словари). Формат JSON был выбран для хранения задач, так как он читается как человеком, так и программой и поддерживается стандартным модулем json из библиотеки Python. Такой подход позволяет легко расширить проект в будущем: например, добавить фильтрацию задач по статусу или приоритету, сортировку, поиск по ключевым словам или даже графический интерфейс.

Проект показал, как можно разделить программу на логические части с помощью функций: загрузка и сохранение данных, обработка ввода пользователя, вывод информации и реализация основной логики в функции main. Это упрощает тестирование и дальнейшую доработку кода. В перспективе приложение можно перенести в веб-интерфейс или Telegram-бота, используя ту же структуру данных и перерабатывая только слой взаимодействия с пользователем.
